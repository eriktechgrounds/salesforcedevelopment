Twee of meer parameters kunnen aan een functie meegegeven worden gescheiden door een komma, bijv function funct(param1, param2, param3)

Door middel van rest parameters kan er een onbeperkt aantal arguments vertegenwoordigd worden als array (een rij bestaande uit elementen). 

    function funct(param1, param2, ...args) {
        //
    }
Hier is de laatste parameter, die voorafgaand met drie punten, de rest parameter (...args). Alle arguments die aan de functie worden meegegeven worden onderdeel van de parameter-lijst. In het voorbeeld hierboven zou een eerste argument gekoppeld worden aan a, het tweede argument aan b en het derde en alle erop volgende aan de rest parameter args, als een array. In dit voorbeeld:
funct(1, 2, 3, 'A', 'B', 'C', 'D')
zal het args-array dus de volgende waardes opslaan: [3, 'A', 'B', 'C', 'D']
Aan een rest parameter kan een oneindig aantal arguments gekoppeld worden, bij expliciete parameters vallen alle arguments groter in aantal dan de expliciet genoemde, "buiten de boot" en worden dus genegeerd.
