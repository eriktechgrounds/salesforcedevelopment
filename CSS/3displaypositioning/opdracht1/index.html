Er bestaan de volgende position properties:


Static: standaardwaarde. Elementen worden in volgorde van de document-flow weergegeven

Absolute: Het element wordt gepositioneerd ten opzichte van zijn eerste gepositioneerde (non-static) ancestor element

Fixed: Het element wordt gepositioneerd ten opzichte van het browservenster

Relative: Het element wordt gepositioneerd ten opzichte van zijn normale positie, dus "left:20px" voegt 20 pixels toe aan de linker kant van het element

Sticky: Het element wordt gepositioneerd op basis van de scroll-positie van de gebruiker
Een sticky element wisselt tussen relative en fixed, afhankelijk van de scroll positie. Het wordt relatief gepositioneerd totdat een bepaalde
offset-positie in de viewport wordt bereikt - dan "plakt" het op zijn plaats (zoals position:fixed).

Initial: Stelt deze eigenschap in op de standaardwaarde.

Inherit: Erft deze eigenschap van zijn parent element.


De default position in HTML is Static. In deze positie zijn de top, bottom, left, en right eigenschappen niet van toepassing.
