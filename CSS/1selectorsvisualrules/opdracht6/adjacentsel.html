De + combinator selecteert naastliggende child elementen.
Dit betekent dat het tweede element direct volgt op het eerste, en dat beide dezelfde parent hebben.
Syntaxis: A + B
Voorbeeld: h2 + p zal overeenkomen met alle <p> elementen die direct volgen op een <h2>. 

        h2 + p {
          background-color: yellow;
        }
