1. Externe CSS: Hierbij word naar een extern bestand verwezen waar vanuit de CSS-code wordt geladen.
Dit is de meest heldere, overzichtelijke manier.

2. Interne CSS: Hierbij wordt binnen het <head> onderdeel van een pagina via een <style>-tag de CSS aan het begin van het document geplaatst.

3. Inline CSS: Hierbij wordt de CSS-code "ter plekke", in een HTML-tag op de pagina toegepast.
