Met een type selector worden alle elementen met eenzelfde node-naam, dus van hetzelfde type, geselecteerd. Bijv: 

        /* Alle <p> elementen worden rood weergegeven. */
        p {
          color: red;
        }
Een descendant selector doelt op alle onderliggende elementen van een bepaald parent element. Bijv: 

        /* Alle <span> elementen die binnen een <div> parent-element vallen worden rood weergegeven. */
        div span {
          color: red;
        }
Een class-selector kiest alle elementen van een bepaalde klasse. Wanneer je dus elementen indeelt in dezelfde klasse, kunnen dezen samen worden aangesproken. Bijv: 

            .city {
                background-color: tomato;
                color: white;
              }
is van toepassing op allebei de volgende elementen, omdat ze in dezelfde "city" klasse zijn ingedeeld: 

  <div class="city">
  <h2>London</h2>
  <p>London is the capital of England.</p>
</div>

<div class="city">
  <h2>Paris</h2>
  <p>Paris is the capital of France.</p>
</div>
