De CSS syntax is als volgt: selector { property: value }
Dmv. het selector element word het specifieke deel code gekozen waarop de volgende CSS code wordt toegepast.
Het property element duidt aan welke eigenschap er wordt gewijzigd.
Het value element geeft de waarde van property aan, waarmee deze nader wordt geduid aan de hand van een specificatie (bijv. aantal, kleur, richting)
